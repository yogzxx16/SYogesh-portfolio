Pulse-Link – Emergency Blood Donor Matching

Tech Stack: HTML, CSS, JavaScript

Pulse-Link is a full-stack prototype designed to demonstrate how an emergency blood donor matching system could work. The project was developed with the help of AI-assisted tools and refined through hands-on implementation and testing.

Key Features:
  • Emergency-focused blood donor matching concept
